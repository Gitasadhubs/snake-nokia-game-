
# Retro Nokia Snake

A high-fidelity Nokia 3310-inspired Snake game featuring classic LCD aesthetics, responsive controls, and a system-driven Game Master for strategy and commentary.

## Features

- **Classic LCD Visuals**: Authentic monochrome screen with pixel-perfect rendering and backlight effects.
- **Physical Phone Interface**: Virtual Nokia 3310 frame with functional D-pad and navigation buttons.
- **Game Master System**: A built-in logic engine that provides witty remarks and strategy tips based on your performance.
- **High Score Tracking**: Persistent high score saved to browser storage.
- **Responsive Controls**: Full support for keyboard (Arrow Keys/WASD/Space) and mobile touch interactions.

## Deployment to Vercel

1. **GitHub Setup**: Push these files to a new GitHub repository.
2. **Import Project**: Log in to [Vercel](https://vercel.com) and click "Add New Project", then select your repository.
3. **Automatic Detection**: Vercel will automatically detect the **Vite** framework preset.
4. **Environment Configuration**: 
   - Ensure any required security keys are added in the "Environment Variables" section of the project settings if applicable.
5. **Deploy**: Click "Deploy" to launch your site.

## Local Development

To run the game locally on your machine:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
